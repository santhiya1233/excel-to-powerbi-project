# Hospital Performance Analysis Dashboard

## Overview
Power BI dashboard analyzing hospital performance including patient admissions, revenue, and department-wise analysis.

## Key Insights
- Total Patients: 200
- Total Admissions: 500
- Revenue: 22.85M
- Avg Treatment Cost: 30.39K

## Features
- Month-wise admission trends
- Department-wise revenue analysis
- City-wise patient distribution
- Doctor-wise patient handling
- Gender-wise admission breakdown

## Tools Used
- Power BI
- DAX
- Data Modeling
