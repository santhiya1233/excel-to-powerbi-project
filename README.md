# Sales and Profit Performance Analysis – Power BI Dashboard

An interactive Power BI dashboard built using the Superstore dataset to analyze sales and profit performance across regions, categories, customer segments, and time.

## 📊 Overview

This project transforms raw Excel (Superstore) data into a fully interactive Power BI report, providing actionable insights into sales trends, profitability, and customer behavior.

## 🔑 Key Metrics (KPIs)

- Total Orders: 4K
- Total Sales: 53,216K
- Total Profit: 10,576K
- Total Quantity: 23K

## 📈 Dashboard Features

- Region Wise Sales – Bar chart comparing sales across East, West, South, and North regions
- Sum of Sales and Profit by Category – Office Supplies, Furniture, and Technology breakdown
- Sum of Sales and Profit by Month – Monthly trend analysis using combo chart
- Sum of Sales by Customer Segment – Donut chart for Home Office, Corporate, and Consumer segments
- Sum of Profit by Customer Type – Treemap view of Regular, Premium, New, and Returning customers
- Filters: Profit Level (Good Profit / High Profit) and Sales Level (Good Sale / High Sale)

## 🛠️ Tools Used

- Microsoft Excel – Data source (Superstore dataset)
- Power BI Desktop – Data modeling, DAX calculations, and dashboard design
- Power Query – Data cleaning and transformation

## 📂 Project Files

- Superstore_Data.xlsx (Project.xlsx)– Raw dataset
- Sales_and_Profit_Performance_Analysis.pbix – Power BI report file

## 🚀 How to Use

1. Clone or download this repository
2. Open the .pbix file in Power BI Desktop
3. Explore the report using filters and slicers (Profit Level, Sales Level)
4. Interact with visuals to drill down into region, category, and customer insights

## 📌 Insights

- Sales are fairly balanced across all four regions
- Customer segments contribute almost equally to total sales (~33% each)
- Profit by customer type is evenly distributed among Regular, Premium, New, and Returning customers
- Monthly sales show seasonal fluctuations with peak months visible in the trend chart
